Инструкция

есть проект java, где есть следующие классы Сards, GsonParser, Main, Root

а так же файл test.json откуда мы берем данные для парсинга(Имя Владельца, Номер карты, Банковская Система и Баланс).


Папку untitled, надо открыть с помощью программы Intellij idea, данная программа парсит json с помощью библиотеки GSON в фреймворке Maven.

В данном коде программа выводит значение объектов, а именно имена владельцев карт и номера.
